#### padStart() padEnd()

ES8  第一个参数是目标长度,第二个参数是要覆盖的字符串

**padStart()**

方法用另一个字符串填充当前字符串(重复，如果需要的话)，以便产生的字符串达到给定的长度。填充从当前字符串的开始(左侧)覆盖。
```
```

**padEnd()**

方法会用一个字符串填充当前字符串（如果需要的话则重复填充），返回填充后达到指定长度的字符串。从当前字符串的末尾（右侧）开始填充。
```
```
